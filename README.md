这是一个基于**菲利普·科特勒《营销革命5.0》**方法论的OpenClaw Skill，经过89轮对抗性迭代优化，专注于"类人技术"赋能营销的策划、诊断与优化。

## 🎯 适用场景

| 任务类型 | 使用模式 |
|---------|---------|
| 帮我写一个X产品/品牌的营销策划方案 | **策划模式** |
| 这个策划方案你看有没有问题 | **评估模式** |

| 我们公司遇到Y问题，怎么破局 | **谋划模式** |

---

## 📊 核心能力

### 策划模式
- 输出营销5.0版策划案框架
- 包含市场背景诊断、技术赋能策略、全渠道体验设计、人机分工、5A旅程覆盖、敏捷验证计划
- **核心技术手段匹配**：预测性营销 / 情境化营销 / 增强现实营销

### 评估模式
- 逐层诊断策划案（框架层→策略层→执行层）
- 五层诊断体系：前提条件、技术匹配、人机分工、全渠道闭环、执行可行性
- 输出可量化的营销5.0味评分

### 修改模式
- 三级修改法（框架层 / 策略层 / 执行层）
- 框架层问题必须重做，策略层局部调整，执行层细节优化

### 谋划模式
- 诊断企业所处营销阶段（1.0-5.0）
- 识别核心痛点，对标营销5.0五大技术手段
- 给出最大杠杆破局方向 + 里程碑验证节点

## 🔧 营销5.0技术手段

| 技术手段 | 解决的问题 | 前置条件 |
|---------|---------|---------|
| 预测性营销 | 新品失败率高、不知道客户下一步要什么 | 6-12个月历史数据 |
| 情境化营销 | 渠道割裂、体验不一致 | IoT传感器/数字化触点 |
| 增强现实营销 | 产品难展示、客户不信任 | 用户数字成熟度高 |
| 对话式营销 | 客服成本高、响应慢 | 有足够咨询量 |
| 敏捷营销 | 市场反应慢、窗口期短 | 组织接受快速迭代 |

## 📖 营销5.0核心框架速查

### 三大社会背景
1. **消费者代沟**：五世同堂（婴儿潮-X-Y-Z-α），需求高度分裂
2. **贫富两极化**：消费主义 vs 极简主义，中产阶级萎缩
3. **数字鸿沟**：技术赋能者 vs 技术落伍者差距拉大

### 五大构成要素
```
数据驱动型营销（基础）
    ↓
预测性营销 ─┬─ 情境化营销 ─┬─ 增强现实营销
（三大核心技术）  (次级支撑)
    ↓
敏捷营销（执行保障）
```

### 营销1.0→5.0演进
| 阶段 | 核心 | 主导逻辑 |
|------|------|---------|
| 1.0 | 产品中心 | 功能差异化 |
| 2.0 | 消费者导向 | 品牌情感诉求 |
| 3.0 | 人文精神 | 价值观共鸣 |
| 4.0 | 数字化 | 与消费者共创 |
| 5.0 | 类人技术 | 以人为本的技术赋能 |

## ⚠️ 核心原则

> **技术是手段，人是核心。**
> AI能发现海量数据中的隐藏模式，但无法替代人类对消费者行为动机的深层理解。营销人员的判断力在复杂情境下仍不可替代。

> **营销5.0是范式转移，不是降本增效工具。**
> 营销4.0是"数字化营销"（提升效率），营销5.0是"数智化营销"（改变竞争优势的根本路径）。
